Mathematics Quiz
Java Game
This project was the last programming assignment of my first year at university. I had to create a game with a graphical user interface. I decided to create a simple mathematics game where the program generates 10 questions, in which the user has to answer each question correctly. I also added individual players to the game, so that players can compete and compare their scores. I was awarded 98% for this assignment, which I am very proud of.
